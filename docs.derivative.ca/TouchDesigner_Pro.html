TouchDesigner Pro[edit]

TouchDesigner Pro includes everything TouchDesigner has to offer.

Pro Support[edit]

Purchasing a Pro license gets you into our Pro Support program. Our team is available to help you with any programming, production, or consulting needs you have. Use our years of experience to make your life easier.

Each Pro license includes 6 hours of Pro Support. Additional hours can be purchased for larger tasks and additional support. Please read Pro Support for more details on what the program includes.

Features Unlocked[edit]

  • Notch operators without any resolution limits (requires Notch license)
  • 3rd party Projection calibration via Vioso TOP or Scalable Display TOP
  • Stype optical tracking for cameras
  • for tracking data from BlackTrax Motion Tracking Systems: BlackTrax CHOP
  • sync multiple processes or systems with Sync In CHOP and Sync Out CHOP
  • hardware frame-lock (NVIDIA Gsync / AMD S400 sync cards)
  • Privacy Options for private .toe files and private components. Create a .toe project file or .tox component file that is inaccessible without the password. Additionally you can create components that only run with USB License Dongle. See Privacy for more details.
  • for blending motion channels: Clip Blender CHOP, Clip CHOP, Clip DAT
  • bug fixes to the specific build you are using
  • software fixes/RFEs are not guaranteed, but Pro users will receive front-of-the-line service.
  • Pro Support program offering direct access to Derivative's programmers and product specialists.

For a comparison of features, see Licensing.
