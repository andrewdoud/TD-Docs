Apple ProRes

From Derivative
Jump to navigation Jump to search

Apple ProRes is a format with high image quality and excellent random access performance. It is less intensive to decode than H.26x codecs but can produce larger file sizes.

TouchDesigner supports decoding and playback of ProRes in the Movie File In TOP, and encoding of ProRes files via the Movie File Out TOP. These operations are hardware accelerated on macOS (Turn on the Movie File In TOP's 'Hardware Decode' parameter toggle on the Tune page). There is no support for hardware decode/encode on Windows.

TouchDesigner supports all 6 standard ProRes formats which you can read about here About Apple ProRes.

All ProRes422-variants are 4:2:2 at 10-bit color depth whereas ProRes 4444 and 4444 XQ is 4:4:4 schema with a color depth of 10 or 12 bits. ProRes 4444 and 4444XQ has the option to include an alpha channel.

TouchDesigner does not currently support Apple ProRes RAW.
